pi100
=====

Using a raspbery pi to do 0-100kph timing via an ODBC connector.

Requires
========

* Pi with Wheezy
* pyserial
* Festical for TTS (sudo apt-get install festival)
* ODBC connector: http://www.sparkfun.com/products/9555
